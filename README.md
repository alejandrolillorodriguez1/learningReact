# Learning React

Proyecto de aprendizaje desarrollado con React para practicar conceptos básicos de frontend moderno.

## Descripción

Este repositorio recoge ejercicios y pruebas realizadas durante el aprendizaje de React. El objetivo principal fue entender cómo funcionan los componentes, el estado, la estructura de una aplicación frontend y la separación de responsabilidades dentro de un proyecto React.

## Tecnologías utilizadas

- React
- JavaScript
- HTML
- CSS

## Conceptos trabajados

- Creación de componentes
- Organización de carpetas
- Uso de props
- Gestión básica del estado
- Maquetación con CSS
- Renderizado dinámico de contenido

## Objetivo del proyecto

El objetivo de este proyecto fue adquirir una base práctica en React y familiarizarme con el desarrollo de interfaces modernas.
